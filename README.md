**Description:**  
Amal is a mobile app that helps patients take medications correctly, avoid errors, and identify early health risks. Built on IBM Watsonx Orchestrate, it automates reminders, tracks adherence, and sends caregiver alerts when needed.

**Features:**  
- Medication dashboard  
- Dose reminders & alerts  
- Early-risk identification  
- Caregiver notifications  

**Technology:**  
- IBM Watsonx Orchestrate  
- Mobile-friendly UI  
- AI-driven workflow  

**How to Run:**  
1. Clone repo  
2. Open in compatible environment  
3. Follow instructions to simulate medication logging & alerts  

**Note:** This is a proof-of-concept project for the Agentic AI Hackathon on Lablab.ai.
